Thank you for using TExDump.

If this is your first time using TexDump, enter your information and click "Register"

If this is not the first computer you're installing TexDump on, enter your information and click "Log In"

For maximum convenience, it is recommended to set TexDump to run on startup.


Hotkey: Ctrl-Shift-C

Using this hotkey will take highlighted text, and send it to whatever machine is currently selected in the GUI.

Click on the icon to bring up the GUI.

Right-Click on the icon to close TexDump.
